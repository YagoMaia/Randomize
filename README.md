# Randomize

```
 ____                 _                 _         
|  _ \ __ _ _ __   __| | ___  _ __ ___ (_)_______ 
| |_) / _` | '_ \ / _` |/ _ \| '_ ` _ \| |_  / _ \

```


1 - You insert how much list for create


2 - You put the elements in the lists


3 - The program will place the elements in random order
